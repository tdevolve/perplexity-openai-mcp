# Security Policy

## Supported Versions

This project currently supports the latest code on the `main` branch only.

## Reporting a Vulnerability

If you believe you have found a security issue, please do not open a public issue with exploit details, credentials, or exposed secrets.

Use GitHub private vulnerability reporting for this repository if available. If private reporting is not available, contact the maintainer directly with the details.

Please include:
- A short description of the issue
- Steps to reproduce it
- The potential impact
- Any suggested fix or mitigation

I will review reports as quickly as practical and work to validate and remediate legitimate issues.
